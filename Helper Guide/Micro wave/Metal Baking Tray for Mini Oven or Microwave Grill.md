```
Comment tu nommerais cette plaque compatible pour supporter le micro onde quand on met des pizzas ou petits four dedans ?
```

Le nom le plus juste pour ça, c’est :

## **Plaque de cuisson métallique rectangulaire**

ou plus précisément :

- **Plaque de four**
- **Plat à four en métal**
- **Lèchefrite** (si elle sert à récupérer jus/graisse)
- **Plaque pour mini-four** (dans ton contexte)

------

## ⚠️ Point important (très important)

Si ton appareil est un **micro-ondes classique (sans mode grill/chaleur tournante)**, **cette plaque en métal ne doit pas être utilisée dedans** → risque d’**arcs électriques**.

Tu peux l’utiliser **seulement** si ton appareil est un :

- **micro-ondes grill**
- **micro-ondes combiné (grill + convection / chaleur tournante)**

et **uniquement** selon le mode autorisé dans le manuel (ex: mode grill/convection, pas mode micro-ondes pur).
